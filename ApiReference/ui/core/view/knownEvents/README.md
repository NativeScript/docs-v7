## Namespace: "ui/core/view".knownEvents
Defines an enum with events for view class.

##### Variables
 - **loaded** - _String_.    
  Raised when the view is added to visual tree and loaded (shown).
 - **unloaded** - _String_.    
  Raised when the view is unloaded.
