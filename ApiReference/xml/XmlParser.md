## Class: "xml".XmlParser  
A simple non-validating SAX parser based on https://github.com/vflash/easysax version 0.1.14

##### Instance Functions
 - **parse(** xmlString _String_ **)**  
     Parses the supplied xml string.
   - **xmlString** - _String_  
     The string containing the xml to parse.
